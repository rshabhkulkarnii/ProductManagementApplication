# Demo Videos

This folder contains demonstration videos of the Product Management Application features.

## Video List

### 1. Application Overview
- **File:** `application-overview.mp4`
- **Duration:** ~5 minutes
- **Description:** Overview of the complete application with all features demonstrated

### 2. CRUD Operations
- **File:** `crud-operations.mp4`
- **Duration:** ~3 minutes
- **Description:** Creating, reading, updating, and deleting products

### 3. Advanced Features
- **File:** `advanced-features.mp4`
- **Duration:** ~4 minutes
- **Description:** Bulk actions, export/import, sorting, and filtering

### 4. UI Features
- **File:** `ui-features.mp4`
- **Duration:** ~3 minutes
- **Description:** Dark/Light theme toggle, statistics dashboard, and notifications

## How to Watch

Simply download the videos from this folder and play them with your favorite video player, or watch them directly on GitHub by clicking on the video files.

## Adding New Demos

To add a new demo video:
1. Record your video (recommended: MP4 format, 1080p or higher)
2. Save it to this `demos/` folder
3. Update this `DEMO.md` file with the new video information
4. Commit and push to GitHub
